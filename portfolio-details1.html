IOT based Realtime Vechicle Management

Project information

  • Category: IOT | Embedded
  • Client: YasinSakib
  • Project date: September, 2020

This project focuses creating a regisrty of vehicles. Through the registration number assigned to each vehicle, the realtime monitoring of vechicle can be achieved, including speed and smoke data. By anlaysis the speed and smoke data, a fitness of the vehicle is calculated. There is also a accident detection system is developed through which the owner get call for any accident should happen.
