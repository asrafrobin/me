A low cost Snacks Vending Machine

Project information

  • Category: Robotics | Automation
  • Developer: Asraful Alam & Russel Rahim
  • Project date: November, 2018

This is a low cost impelementaion of snacks vending machine with all fuctionallities that of a commercial vending machine. This two shelfed vending machine has six (6) spring dispenser, each of which can carry upto four packets of snacks. Besides, the intutive user interface allows an user to order desired number of snacks and put coins in the coin acceptor and in turn the dispenser dispatch the required amount of snacks.
